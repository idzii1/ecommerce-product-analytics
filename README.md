# E-Commerce Sales Analytics Dashboard

## Business Problem
The goal of this project was to analyze e-commerce sales data to understand revenue drivers and customer behavior.

## Tools
- SQL (SQLite)
- Python (pandas, matplotlib)
- Power BI

## Analysis
- Revenue by category
- Monthly sales trends
- Customer segmentation
- KPI metrics

## Key Insights
- Electronics generates the highest revenue
- Sales trend shows consistent growth
- A small number of customers contribute significantly to total revenue

## Business Recommendation
- Focus on top-performing categories
- Implement retention strategies for high-value customers
